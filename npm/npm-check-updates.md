# With `npm` OR `yarn` check updates

## npm

```shell
# npm-check

npx npm-check -u

# npm-upgrade

npx npm-upgrade

# npm-check-updates

npx npm-check-updates

```

## yarn

```shell
yarn upgrade-interactive --latest
```
